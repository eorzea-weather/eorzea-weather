# eorzea-time

:clock3: A clock of the Eorzea time

## Install

```shell
npm install --save eorzea-time
```

## Usage

### ES2015

```javascript
import EorzeaTime from 'eorzea-time';

const eorzeaTime = new EorzeaTime();
console.log(eorzeaTime.toString);
```
